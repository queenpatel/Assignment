(1) What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library that makes things like HTML document traversal and manipulation, event handling, animation, and AJAX much simpler with an easy-to-use API that works across a multitude of browsers. It's often described as a "write less, do more" library, because it takes a lot of common tasks that require many lines of JavaScript code to accomplish and wraps them into methods that you can call with a single line of code. This makes it much easier to develop interactive web applications.

(2) How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

1. Using the css() method:
Directly set CSS properties for selected elements:
$(selector).css({ property1: value1, property2: value2, // ... }); // Example: $("p").css({ "color": "red", "font-size": "20px" });

2. Adding and removing classes:
addClass(className): Adds one or more classes to selected elements. removeClass(className): Removes one or more classes from selected elements. toggleClass(className): Toggles the presence of a class on selected elements (adds it if absent, removes it if present). // Add a class: $("p").addClass("highlight"); // Remove a class: $("p").removeClass("highlight"); // Toggle a class: $("button").click(function() { $("p").toggleClass("hidden"); });

3.jQuery animations:
jQuery offers various built-in effects for animations: $(selector).hide(speed, callback); // Hide elements $(selector).show(speed, callback); // Show elements $(selector).toggle(speed, callback); // Toggle visibility $(selector).fadeIn(speed, callback); // Fade elements in $(selector).fadeOut(speed, callback); // Fade elements out $(selector).fadeToggle(speed, callback); // Toggle fade $(selector).slideDown(speed, callback); // Slide elements down $(selector).slideUp(speed, callback); // Slide elements up $(selector).slideToggle(speed, callback); // Toggle slide
