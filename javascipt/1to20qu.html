Q10: What is the result of the expression (5 > 3 && 2 < 4)?

5 > 3 evaluates to true because 5 is greater than 3. 2 < 4 evaluates to true because 2 is less than 4. The logical AND operator (&&) returns true if both operands are true. 5 > 3 and 2 < 4 are true, the combined expression (5 > 3 && 2 < 4) evaluates to true. result of the expression (5 > 3 && 2 < 4) is true.

Q11: What is the result of the expression (true && 1 && "hello")?

The expression (true && 1 && "hello") uses the logical AND operator (&&) to combine three operands: a boolean, a number, and a string. In JavaScript, the && operator evaluates from left to right and returns the first falsy value it encounters. If all values are truthy, it returns the last value. 1. true && 1: true is a truthy value. 1 is also a truthy value. Since both are truthy, this part of the expression evaluates to 1. 1 && "hello": 2. 1 is a truthy value. "hello" is also a truthy value. Since both are truthy, this part of the expression evaluates to "hello". Therefore, the result of the expression (true && 1 && "hello") is "hello".
